# 阅读专注力评估系统

## 项目概述

通过同步记录用户的朗读声音和手指滑动轨迹，评估小孩阅读专注力。

## 核心功能

1. **语音识别**：实时将用户朗读转为文字
2. **滑动跟踪**：记录鼠标在横线上的滑动进度
3. **智能比对**：当阅读完成100字时，比对语音识别结果与滑动覆盖的文本
4. **结果评估**：计算阅读匹配度和专注力得分

## 使用方法

```bash
cd ~/Documents/claude/reading_focus_demo

# 启动简单HTTP服务器
python3 -m http.server 8080
# 或
npx --yes serve
```

然后访问 http://localhost:8080

## 技术栈

- 前端：原生 HTML/CSS/JavaScript
- 语音识别：Web Speech API
- 文本匹配：编辑距离算法

## 工作流程

1. 用户点击"开始阅读"按钮
2. 系统开始录音并显示文本
3. 用户一边朗读，一边用鼠标在横线上滑动
4. 系统实时记录：
   - 语音识别结果
   - 鼠标滑动覆盖的文本范围
5. 当阅读满100字时，系统自动进行比对分析
6. 显示评估结果
