# 阅读专注力评估 - 简化版

## 需求
只处理一句话：`春眠不觉晓、处处闻啼鸟`

## 任务拆分

### 任务1：界面布局
- [ ] 显示目标文本：`春眠不觉晓、处处闻啼鸟`
- [ ] 在文字下方绘制下划线区域
- [ ] 添加"开始阅读"按钮
- [ ] 添加"结束阅读"按钮

### 任务2：录音功能
- [ ] 使用 Web Speech API 实现语音识别
- [ ] 点击"开始阅读"时启动录音
- [ ] 点击"结束阅读"时停止录音
- [ ] 获取完整的语音识别文字

### 任务3：鼠标滑动位置检测
- [ ] 在下划线区域监听鼠标移动/点击事件
- [ ] 计算鼠标X坐标对应第几个字
- [ ] 记录鼠标最终停留位置的字数

### 任务4：对比与计分
- [ ] 点击"结束阅读"时，获取语音识别结果
- [ ] 获取鼠标停留位置前的所有文字（前N个字）
- [ ] 对比两个文字是否相同
- [ ] 相同显示100分，不同显示0分

## 流程图
```
开始阅读 ──→ 录音中 + 记录鼠标位置 ──→ 结束阅读
                      ↓
              鼠标停留在第X个字
                      ↓
              对比：语音文字 vs 前X个字
                      ↓
              相同=100分 ≠0分
```

## 技术实现要点

1. **鼠标位置转字数**：
   - 文字区域总宽度 / 总字数 = 每个字的宽度
   - 鼠标X位置 / 每个字宽度 = 对应第几个字

2. **语音识别**：
   - 使用 Web Speech API (仅 Chrome 支持)
   - 需要用户授权麦克风

3. **对比逻辑**：
   - 语音识别结果应该朗读："春眠不觉晓处处闻啼鸟"（去标点）
   - 鼠标停留位置前文字："春眠不觉晓"（前5个字）
